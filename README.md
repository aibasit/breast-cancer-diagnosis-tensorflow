# 🧬 Breast Cancer Diagnosis using TensorFlow

This project presents a machine learning solution for breast cancer diagnosis using the **UCI Breast Cancer Wisconsin (Diagnostic) dataset**. The objective is to classify tumors as **malignant** or **benign** using a TensorFlow-based neural network. The notebook demonstrates a complete workflow from raw data to model evaluation, ensuring clarity and reproducibility.

### 🚀 Key Components
- Data loading and preprocessing  
- Feature normalization and scaling  
- Train-test split for model validation  
- Neural network model built using TensorFlow  
- Model training and performance evaluation  

### 📊 Outcome
The model effectively learns patterns from medical data to perform binary classification. This project highlights how deep learning techniques can be applied to real-world healthcare problems, enabling early detection and decision support systems.

### 🛠️ Tech Stack
- Python  
- TensorFlow  
- NumPy, Pandas  
- Matplotlib / Visualization tools  

This implementation focuses on simplicity, interpretability, and practical understanding of machine learning workflows, making it suitable for both academic purposes and foundational AI applications in healthcare.
